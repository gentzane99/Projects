# Agoodshop – Product Detail Page

Maqueta responsive de la página de detalle de un producto para una tienda online ficticia llamada **Agoodshop**.  
Proyecto realizado como actividad académica para practicar maquetación web con HTML y CSS.

## Objetivo del proyecto
Diseñar y maquetar una ficha de producto similar a la de un e-commerce real, prestando especial atención a:
- La estructura del contenido
- El uso correcto de HTML semántico
- El diseño responsive para móvil y escritorio

## Tecnologías utilizadas
- HTML5
- CSS3
- Responsive Design (mobile & desktop)

## Características
- Maqueta basada en un diseño de referencia
- Adaptación responsive en un único archivo HTML
- Uso de etiquetas semánticas para mejorar la estructura
- Formulario preparado para ser funcional

## Notas
- El formulario no es funcional, solo estructural.
- El diseño prioriza la correcta organización del contenido y la adaptabilidad frente al detalle visual exacto
